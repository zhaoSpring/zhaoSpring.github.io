函数定义

  • fn关键字声明函数
  • 函数名使用下划线命名法（蛇形命名法，snake_case）：字母小写，下划线分隔单词

示例：

函数参数

  • 必须声明每个参数的类型
  • 参数数量是固定的

函数体的语句和表达式

  • 语句：执行操作但不返回值，语句末尾有;
  • 表达式：计算并返回值，表达式末尾没有;

函数返回值

  • 使用 -> 声明返回类型
  • 隐式返回最后一个表达式的值
  • 也可以使用 return 提前返回指定值
